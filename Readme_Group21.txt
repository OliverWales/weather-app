Before running our application you will need to include the Google GSON library (included in /lib/gson-2.6.2.jar, also available at https://github.com/google/gson).
We have successfully performed testing on Windows, Mac and Linux.
Source code is contained within the src directory and uses data in the data directory.
(The out and .idea directories are used to load our project into IntelliJ)
To execute our program open the project in IntelliJ and execute MainScreen.main()
